💸 Expense Tracker – Personal Cashflow System

A clean, fast and practical cashflow tracking app built with pure JavaScript.
Designed for people who already have money and need clarity, control and decision-making power.

🚀 Live Demo

👉 https://alejocalivari.github.io/expense-tracker/

🧠 Concept

Most expense trackers are made for saving pennies.
This one is built for managing real cashflow:

How much money do I actually have left?
How much can I spend per day?
Am I overspending this month?
How much am I investing vs consuming?
⚙️ Features
💰 Core System
Monthly income input
Expense tracking (categorized)
Real-time balance calculation
“Money available” logic (not just raw balance)
📊 Smart Insights
Daily available money
Monthly flow visualization
Category distribution
Key financial metrics (investor mindset)
🔍 UX Focus
Fast input system
Search bar for movements
Clean dashboard structure
Mobile-first design
🌐 Extra
EN / ES toggle
ARS → USD conversion (base rate configurable)
LocalStorage persistence (no backend needed)
PWA ready (installable app)
📁 Data Portability
Export data to JSON
Import data from JSON
(Optional) CSV support
🛠️ Tech Stack
HTML5
CSS3 (responsive + modern UI)
Vanilla JavaScript (modular logic)
LocalStorage (data persistence)

No frameworks. No dependencies. Pure control.
